# Airbnb-Data-Analysis
# 🏠 Airbnb Data Analysis & Price Prediction

## 📌 Overview
This project analyzes Airbnb listing data to understand pricing trends and build a predictive model.

## 🎯 Objectives
- Perform Exploratory Data Analysis (EDA)
- Analyze price distribution and trends
- Detect and remove outliers
- Perform statistical tests
- Build a Linear Regression model

## 📊 Techniques Used
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- SciPy

## 📈 Visualizations
- Histogram
- Scatter plots
- Heatmap
- Boxplot
- Time series

## 🤖 Machine Learning
- Linear Regression
- Evaluation using MAE, MSE, R²

## 📂 Dataset
Airbnb dataset from Kaggle

## 🚀 Results
- Identified key factors affecting price
- Built a prediction model

## 🧠 Conclusion
This project provides insights into Airbnb pricing patterns and demonstrates data analysis and machine learning skills.
